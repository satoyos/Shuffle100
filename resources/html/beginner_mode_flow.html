1. 序歌の読み上げ

トップ画面で「試合開始」ボタンを押すと、まず序歌が読み上げられます。

序歌の下の句は、2回読まれます。

 

2. 1首目の歌の読み上げ

序歌に続いて、自動的に1首目が読み上げられます。

上の句の読み上げが終わると、少し間を置いて自動的に下の句を読み上げます。

(上の句と下の句の間隔や歌と歌の間隔、読み上げの音量などを調整するには、読み上げ画面のボタンを押してください。)

 

3. 下の句まで読んだ後の選択

下の句の読み上げが終わると、「次はどうする？」画面が表示されます。

今読み終えたばかりの下の句を繰り返し読み上げるのか、あるいは次の歌に進むのか、を選択してください。

 

4. 2首目の歌以降、繰返し

2首目以降も1首目の場合と同様に、歌の上の句と下の句を一通り読み終えたところで試合は一旦区切れますので、「次はどうする？」画面で次の動作を選んでください。
