1. 「札セット」とは？

読み上げる歌を選んだ状態を、アプリが覚えていてくれたら…と思うことはありませんか？

このアプリでは、Ver.5.0から「今選んでいる歌」のセットに名前を付けた「札セット」を保存し、後から呼び出せるようになりました。

「もう覚えた札」セットや、「1字決まりの歌」セットなど、いろんなパターンのセットを作ってみてください。

 

2. 札セットを作る（保存する）

「歌を選ぶ」画面の右上に、「保存」ボタンがあります。

このボタンを押すと、今選んでいる歌のセットに名前を付けて保存することができます。

また、既に保存済みの札セットがある場合には、それを「上書きする」選択肢も現れます。
上書きすることで、その札セットの中身を、今選んでいる歌に書き換えることができます。

 

3. 保存した札セットを呼び出す

保存した札セットを使う（呼び出す）には、「歌を選ぶ」画面右下の「まとめて選ぶ」を押して…

表示されるメニューから「作った札セットから選ぶ」をタップしてください。

すると、これまでに作った札セットの一覧が表示されますので、呼び出したい札セットを選んでください。

なお、保存している札セットが無い状態では、「まとめて選ぶ」を押したときのメニューに「作った札セットから選ぶ」選択肢は現れません。

 

4. 札セットを削除する

不要になった札セットを消したい場合には、「保存した札セットを呼び出す」ときと同じ手順で札セット一覧画面を表示して、消したい札セットを左にスワイプしてください。

ここで「削除」を押すと、その札セットを消すことができます。
